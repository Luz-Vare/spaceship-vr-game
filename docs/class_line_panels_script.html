Detailed Description

LinePanelsScript es una clase muy similar a BasePanelScript. Esta asociada a cada panel con linea en el medio, detecta cuando pasamos el ratón por encima para enviar una petición de moverse a PanelsManagerScript Además, si se desplaza cambiará el color de la línea dependiendo si está o no en la fila "central"

void LinePanelsScript.moveThisPanel ( )

Cuando pasamos el puntero por encima del panel, se llamam a este método. Envía una petición a PanelsManagerScript para moverse a un hueco cercano, si existiese, dentro del juego. Si consigue moverse reproduce un sonido y actualiza el color de su linea dependiendo si su nueva posición es en la línea central o no
