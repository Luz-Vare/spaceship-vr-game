void Bit3.chageState ( )

Este método comprueba si el juego ha iniciado, si es así, cambia el estado del cubo actual y aumenta la cantidad de pasos que ha dado el usuario, si es mayor que el número mínimo, decrece en 1 unidad el score del jugador. Además reproduce los sonidos correspondientes al cambio de estado y coloca el cubo en el color correspondiente. Este método es llamado por el cubo cuando el usuario lo activa.
